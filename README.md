# HRnet ReactJs App

This is the ReactJs version of the HRnet application.

## Getting Started

1- clone the project

```bash
cd ./yourDirectory
git clone https://github.com/Salim-camara/jqueryToReact
```

2- intall node_modules

```bash
cd ./jqueryToReact
npm install OR yarn
```

3-
to start the project, just run

```bash
npm start OR yarn start
```

Enjoy :=)
